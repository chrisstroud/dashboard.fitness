# Story 8.1: Stack CRUD — Create, Rename, Delete from Context Menus

## User Story

**As a** dashboard user
**I want to** create, rename, and delete habit stacks from context menus
**So that** I can refine my routine structure over time

---

## Acceptance Criteria

- [ ] **AC1:** Section header "…" menu includes "Add Habit Stack" that creates a new stack via alert
- [ ] **AC2:** Stack header has a "…" menu with "Rename", "Delete Stack", and "Add Protocol Here"
- [ ] **AC3:** Creating a stack: alert with name field → `ProtocolGroup` created at bottom of section → synced to server
- [ ] **AC4:** Renaming a stack: alert with pre-filled name → updates group name locally + server
- [ ] **AC5:** Deleting a stack: confirmation dialog → calls `DELETE /api/protocols/groups/{id}` → removes locally → protocols reassigned
- [ ] **AC6:** Deleting the last stack shows an error alert ("Cannot delete the last stack")
- [ ] **AC7:** "Add Protocol Here" opens `CreateProtocolSheet` with `initialSection` + `initialStack` pre-set

## Technical Context

### Architecture Reference

Stack CRUD is local-first for create/rename (write to SwiftData, then sync). Delete is server-first (API handles protocol reassignment atomically, then local refresh).

## Implementation Guidance

### Approach

Add state variables for the create/rename alerts and the delete confirmation. Add a context menu to the `HabitStackHeader` (or wrap it in a contextMenu modifier). Wire handler functions for create, rename, and delete. Create uses the existing `POST /api/protocols/sections/{id}/groups`. Rename uses `PUT /api/protocols/groups/{id}`. Delete uses `DELETE /api/protocols/groups/{id}` (Story 5.2).

5. **Handler functions:**

```swift
// MARK: - Stack CRUD

private func createStack() {
    guard !newStackName.isEmpty, let section = addStackTargetSection else { return }
    let group = ProtocolGroup(name: newStackName, position: section.groups.count)
    group.section = section
    modelContext.insert(group)
    try? modelContext.save()

    // Sync to server
    Task {
        guard let token = AuthService.shared.token else { return }
        #if DEBUG
        let baseURL = "http://localhost:5001"
        #else
        let baseURL = "https://dashboardfitness-production.up.railway.app"
        #endif
        var request = URLRequest(url: URL(string: "\(baseURL)/api/protocols/sections/\(section.id.uuidString)/groups")!)
        request.httpMethod = "POST"
        request.setValue("Bearer \(token)", forHTTPHeaderField: "Authorization")
        request.setValue("application/json", forHTTPHeaderField: "Content-Type")
        request.httpBody = try? JSONSerialization.data(withJSONObject: [
            "id": group.id.uuidString, "name": group.name, "position": group.position,
        ])
        _ = try? await URLSession.shared.data(for: request)
    }
    newStackName = ""
}

private func renameStack() {
    guard !newStackName.isEmpty, let group = renamingStack else { return }
    group.name = newStackName
    renamingStack = nil
    newStackName = ""

    Task {
        guard let token = AuthService.shared.token else { return }
        #if DEBUG
        let baseURL = "http://localhost:5001"
        #else
        let baseURL = "https://dashboardfitness-production.up.railway.app"
        #endif
        var request = URLRequest(url: URL(string: "\(baseURL)/api/protocols/groups/\(group.id.uuidString)")!)
        request.httpMethod = "PUT"
        request.setValue("Bearer \(token)", forHTTPHeaderField: "Authorization")
        request.setValue("application/json", forHTTPHeaderField: "Content-Type")
        request.httpBody = try? JSONSerialization.data(withJSONObject: ["name": group.name])
        _ = try? await URLSession.shared.data(for: request)
    }
}

private func deleteStack() {
    guard let group = deleteStackTarget else { return }
    guard let section = group.section, section.groups.count > 1 else {
        deleteError = "Cannot delete the last stack in a section."
        deleteStackTarget = nil
        return
    }

    Task {
        guard let token = AuthService.shared.token else { return }
        #if DEBUG
        let baseURL = "http://localhost:5001"
        #else
        let baseURL = "https://dashboardfitness-production.up.railway.app"
        #endif
        var request = URLRequest(url: URL(string: "\(baseURL)/api/protocols/groups/\(group.id.uuidString)")!)
        request.httpMethod = "DELETE"
        request.setValue("Bearer \(token)", forHTTPHeaderField: "Authorization")
        let (_, response) = try await URLSession.shared.data(for: request)
        if let http = response as? HTTPURLResponse, http.statusCode == 204 {
            await MainActor.run {
                // Refresh from server to get reassigned protocols
                Task { await SyncService.shared.syncAll(modelContext: modelContext) }
            }
        } else if let http = response as? HTTPURLResponse, http.statusCode == 400 {
            await MainActor.run {
                deleteError = "Cannot delete the last stack in a section."
            }
        }
    }
    deleteStackTarget = nil
}
```

### Gotchas / Watch Out For

- **Delete is server-first:** Don't delete the local group — let `syncAll` refresh the state after the server handles protocol reassignment. This avoids needing to replicate the reassignment logic on the client.
- **Alert stacking:** SwiftUI allows multiple `.alert()` modifiers on the same view. Each is keyed by its `isPresented` binding. Don't worry about conflicts.
- **`CreateProtocolSheet` `initialStack`:** The "Add Protocol Here" button on a stack menu needs to pass `initialStack`. This requires `CreateProtocolSheet` to accept that parameter (Story 6.4). If 6.4 isn't done yet, pass `initialSection` only.
